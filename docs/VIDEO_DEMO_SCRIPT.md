## üé• TECHNICAL FILMING NOTES

### Setup Requirements
- **Screen recording**: 1920x1080 resolution minimum
- **Audio**: Clear voiceover with minimal background noise
- **Cursor**: Use a tool to highlight cursor for visibility
- **Transitions**: Smooth, professional (fade or slide)
- **Pacing**: Not too fast - let viewers absorb information

### Editing Tips
1. **Add timestamps** in description for each section
2. **Use callout boxes** to highlight key metrics
3. **Zoom in** on important UI elements (badges, citations, confidence scores)
4. **Add subtle music** (optional) - professional, not distracting
5. **Include captions** for accessibility

### Pre-Recording Checklist
- [ ] System is running with all services healthy
- [ ] 100+ regulations loaded and searchable
- [ ] Sample queries prepared and tested
- [ ] Compliance form data ready to input
- [ ] Browser zoom set to comfortable viewing level (100-110%)
- [ ] Dark mode decision made (light mode recommended for clarity)
- [ ] Test run completed to verify timing

- **Opening**: 0:00 - 0:20 (20s)
- **Section 1 (Impact/Interop/Scale)**: 0:20 - 1:20 (60s)
